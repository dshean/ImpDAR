impproc¶

An executable to perform single processing steps

usage: impproc.py [-h]
                  {hfilt,ahfilt,rev,cat,restack,rgain,agc,vbp,crop,nmo} ...

Sub-commands:¶

hfilt¶

Horizontally filter the data by subtracting the average trace from a window

impproc.py hfilt [-h] [-o O] [-pe] [-gssi] start_trace end_trace fns [fns ...]

Positional Arguments¶

start_trace First trace of representative subset
end_trace Last trace of representative subset
fns The files to process

Named Arguments¶

-o Output to this file (or folder if multiple inputs)
-pe

Inputs are pulse ekko files

Default: False

-gssi

Inputs are gssi files

Default: False

crop¶

Crop the data in the vertical

impproc.py crop [-h] [-o O] [-pe] [-gssi]
                {top,bottom} {snum,twtt,depth} lim fns [fns ...]

Positional Arguments¶

top_or_bottom

Possible choices: top, bottom

Remove from the top or bottom

dimension

Possible choices: snum, twtt, depth

Set the bound in terms of snum (sample number), twtt (two way travel time in microseconds), or depth (m, calculated using the nmo_depth or a light speed of 1.69e8m/s if it doesn’t

lim The cutoff value
fns The files to process

Named Arguments¶

-o Output to this file (or folder if multiple inputs)
-pe

Inputs are pulse ekko files

Default: False

-gssi

Inputs are gssi files

Default: False

nmo¶

Normal move-out correction

impproc.py nmo [-h] [--uice UICE] [--uair UAIR] [-o O] [-pe] [-gssi]
               ant_sep fns [fns ...]

Positional Arguments¶

ant_sep Antenna separation
fns The files to process

Named Arguments¶

–uice

Speed of light in ice in m/s (default 1.69e8)

Default: 169000000.0

–uair

Speed of light in air in m/s (default 3.0e8)

Default: 300000000.0

-o Output to this file (or folder if multiple inputs)
-pe

Inputs are pulse ekko files

Default: False

-gssi

Inputs are gssi files

Default: False
