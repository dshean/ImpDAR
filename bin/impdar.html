impdar¶

The main executable for the ImpDAR package.

usage: impdar.py [-h] {load,proc,plot,convert} ...

Sub-commands:¶

load¶

Load data

impdar.py load [-h] [-o O] {gssi,pe,mat} fn [fn ...]

Positional Arguments¶

filetype

Possible choices: gssi, pe, mat

Type of file

fn File(s) to load

Named Arguments¶

-o Write to this filename

proc¶

Process data

impdar.py proc [-h] [-gssi] [-pe] [-cat] [-vbp VBP VBP] [-hfilt HFILT HFILT]
               [-ahfilt] [-rev] [-nmo NMO NMO] [-crop CROP CROP CROP]
               [-restack RESTACK] [-interp INTERP INTERP] [-o O]
               fn [fn ...]

Positional Arguments¶

fn File(s) to process

Named Arguments¶

-gssi

Indicates that the file(s) are gssi output

Default: False

-pe

Indicates that the file(s) are pulse ekko output

Default: False

-cat

Concatenate the files

Default: False

-vbp Bandpass the data vertically at low (MHz) and high (MHz)
-hfilt Remove the average trace (average between hfilt0 and hfilt1)
-ahfilt

Adaptive horizontal filtering

Default: False

-rev

Reverse profile

Default: False

-nmo Normal moveout correction. First argument is the transmitter-receiver separation. Second argument is the velocity of the radar wave (in m/s).
-crop Crop the radar data in the travel-time direction. Arguments are the limit, whether to crop off [“top”, “bottom”], with limit defined in terms of [“snum”, “twtt”, “depth”]
-restack Restack to this (odd) number of traces
-interp Reinterpolate GPS. First argument is the new spacing, in meters. Second argument is the filename (csv or mat) with the new GPS data
-o Write to this filename

plot¶

Plot data

impdar.py plot [-h] [-s] [-yd] [-xd] [-tr TR TR] [-o O] fn [fn ...]

Positional Arguments¶

fn File(s) to plot

Named Arguments¶

-s

Save file (do not plt.show())

Default: False

-yd

Plot the depth rather than travel time

Default: False

-xd

Plot the distance rather than the trace number

Default: False

-tr Plot the traces in this range (line plot)
-o Write to this filename

convert¶

Convert filetype (potentially lossy)

impdar.py convert [-h] [-in_fmt {mat,gssi,pe}] [-t_srs T_SRS]
                  fn [fn ...] {shp,mat}

Positional Arguments¶

fn File(s) to convert
out_fmt Possible choices: shp, mat

Named Arguments¶

-in_fmt

Possible choices: mat, gssi, pe

Input format type. If none, guess from extension

-t_srs

Target spatial reference system (only used if out_fmt==shp). Give as EPSG number.

Default: 4326
