plot¶

impdar.lib.plot.plot(fn, tr=None, gssi=False, pe=False, s=False, ftype='png', dpi=300, xd=False, yd=False, x_range=(0, -1), *args, **kwargs)¶

We have an overarching function here to handle a number of plot types

Parameters:
  • fn (list of strs) – A list of filenames to plot individually.
  • tr (tuple or int, optional) – Plot traces tr[1] to tr[2] (or trace tr) rather than the radargram. Default is None (plot radargram)
  • gssi (bool, optional) – If True, fns are .DZT files
  • pe (bool, optional) – If True, fns are Pulse Ekko files
  • x_range (tuple, optional) – The range of traces to plot in the radargram. Default is (0, -1) (plot all traces)
